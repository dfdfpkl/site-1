Основы сборки и программирования

Приветствую, дорогой олипиадник! Мы рады видеть тебя на нашем сайте для подготовки к соревнованиям по робототехнике. Здесь ты можешь вспомнить основные детали программирования.

Как задать программу движения?

1. настроить датчики и порты.

2. Использовать программу "void move", подписав в скобках переменные мощности движения и времени в милисекундах.

3. Пример:
{
void move(int v2, int v3, int t);
motor[motor1]=v2;
motor[motor2]=v3;
wait1Msec(t);
}

!ВАЖНО! не забывать прo фигурные скобки!

Как запрограммировать повороты по гидроскопу?

1. Настроить датчики и порты

2. введение гидроскопа через
{
void deq(int deg, int k, int n)
{resetgyro(PortQ)}
}

3. введение переменных

Как запрограммировать пульт?

1. настроить датчики и порты.

2. прописать моторы через void move(int v2, int v3, int t);

3. задать значения переменных.

4. прописать кнопки через переменные, пример: w = vexRT[BtnRup]+2*vexRT[BtnRDown];

5. настроить чувствительность джойстика, пример:
{
case 1:
move_m(0,-127,2)
break;
}
